Simple Platformer Tileset Pixel Art

Description

We publish Simple Platformer Tileset Pixel Art. This collection of graphics is designed to create a variety of levels in 2D video games.

The kit includes:
  • Tiles (32×32) for creating platforms on which the character will move;
  • Objects (trees, barrels, boxes, bushes, stairs, stones, plates, a house, etc.);
  • Animated items (chest, coin, flag, key, star);
  • Background 576×324.

If you create a game in pixel art style, then this graphics may be useful to you. The archive contains the following formats: PSD, PNG.
